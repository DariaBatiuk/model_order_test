# Order Creation Test Task (Flutter)

Тестовое задание: реализация экрана создания заказа в Flutter с использованием REST API.

## Функциональность

Реализовано:

- модель `Order` с `fromJson`
- кастомное исключение `ApiException`
- сервис `OrderService` для отправки POST запроса `/api/orders`
- обработка:
  - успешного ответа (200)
  - ошибок сервера (400+)
  - отсутствия интернета
  - timeout (10 секунд)
- `OrderControllers` со статусами:
  - `initial`
  - `loading`
  - `success`
  - `error`
- экран создания заказа:
  - кнопка **Create Order**
  - индикатор загрузки
  - отображение ошибки
  - блокировка кнопки во время запроса
  - возможность повторить попытку

## Запуск проекта

1. Установить зависимости
2. Запустить приложение 
flutter run
